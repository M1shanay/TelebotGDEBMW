Для запуска потребуется:

1. Устанавливает Python 3.12
2. Устанавливаем все зависимости из req.txt
3. Устанавливает Tesseract из папки ReqSoft
4. Для просмотра базы данных используем SQLiteStudio из папки ReqSoft

Если Tesseract устанавливается не по стандартному пути, то в файле tesseractocr.py изменяем путь к .exe файлу на свой

После установки всего необоходимого запускаем файл bot.py и переходим на бота по ссылке https://t.me/sbk_gde_bmw_bot
